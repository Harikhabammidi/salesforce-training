This repository contains my Day 2 learning notes on Salesforce Platform basics.

📌 What is Salesforce Platform?
Salesforce Platform is a cloud-based platform used to build and manage business applications using apps, objects, tabs, and automation tools.

🧩 Salesforce Basic Concepts
📱 App
An App is a collection of related features and objects created for a specific purpose.
Example: College Management App

🗂️ Object
An Object stores data in Salesforce like a database table.
Example: Student

📑 Tab
A Tab is used to open and access objects or applications in Salesforce.

🔄 CRM + Platform
CRM data such as Accounts, Contacts, and Opportunities are stored as Objects inside Apps. Users access them through Tabs.

⚙️ Configuration vs Coding
🔹 Configuration (No Code)
Customization using clicks and Salesforce tools.
Examples:


Creating fields


Building workflows



🔹 Coding (Apex)
Used for advanced logic and custom features.
Examples:


Custom automation


Complex validations
